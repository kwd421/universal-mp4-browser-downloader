# ClipFlow 1.0.8

## 변경 사항

### 라이트 / 다크 모드
- 하단 푸터에서 **라이트 ↔ 다크**를 바로 전환할 수 있습니다. (선택값은 다음에 실행할 때도 유지됩니다.)
- 라이트는 **소프트 화이트** 계열(순백 페이지·베이지 일색 UI 대신)로 맞춰, 배경과 버튼·입력칸이 자연스럽게 구분됩니다.
- 다크는 페이지 → 카드 → 입력/버튼 단계가 보이도록 명암을 나눴고, **호버 시 채움·테두리·아이콘이 더 또렷하게** 바뀌도록 조정했습니다.
- 글자·아이콘·테두리는 순흑/순백 대신 조금 부드러운 중립 톤을 써서 장시간 볼 때 덜 부담스럽게 했습니다.

### 컨트롤 크롬 통일
- URL 입력, 저장 폴더, 쿠키, 구간선택·저장 위치 등 **같은 계열 컨트롤의 테두리·채움 색을 한 토큰으로 맞춤**습니다.
- 라이트 테두리는 다시 **검정 계열**로 보이게 정리했습니다. (연한 갈색 테두리로 보이던 문제 수정)
- 다크에서 쿠키 아이콘·메타 아이콘이 라이트 검정으로 굳어 보이던 문제를 고쳤습니다. (테마 전환 시 색을 다시 읽도록 수정)

### 파비콘
- 사이트 파비콘·유튜브 배지는 **테마에 따라 색이 뒤집히지 않습니다.** (원본 브랜드 색 유지)
- 글롭 폴백은 고정 중립 회색입니다.
- 파비콘이 너무 튀지 않도록 **살짝만 투명도**를 낮췄습니다. (색을 탁하게 바꾸지는 않음)

### 기타
- 진행/분석 표시줄 무지개 색도 테마 토큰을 쓰도록 정리했습니다.
- 테마 전환 시 필드·콤보·소스 링크 아이콘이 바로 다시 그려집니다.

## 확인 방법

1. 1.0.7 이하를 실행하면 시작 후 1~2초 뒤 **「1.0.8 업데이트가 있습니다」** 토스트가 뜹니다.
2. 업데이트 후 푸터의 달/해 아이콘으로 라이트·다크를 바꿔 보세요.
3. 다크에서 버튼·쿠키·URL 칸에 마우스를 올리면 호버가 이전보다 분명하게 보여야 합니다.
4. 목록 행의 파비콘은 다크로 바꿔도 사이트 색이 유지되어야 합니다.
