# ClipFlow 1.0.5

## 변경 사항

- 시작할 때 업데이트 확인 때문에 잠깐 **응답 없음**이 뜨던 문제를 수정했습니다.
- 업데이트 알림이 WinSparkle 콜백에만 의존하지 않고, **appcast를 직접 확인**해 토스트를 띄우도록 수정했습니다.
- GitHub Pages appcast가 늦게 반영될 때를 대비해 **백업 appcast URL**도 확인합니다.
- 구간선택 팝업에서 컷 방식 선택 UI를 제거했습니다. 구간 다운로드는 항상 빠른 컷입니다.
- URL 입력 포커스 테두리, 검색 한글 IME를 다듬었습니다.
- 다운로드 카드 커서: 파비콘과 호버 액션 아이콘만 손가락, 나머지는 화살표입니다.

## 업데이트 안내

- **1.0.4**에서 실행하면 시작 후 잠시 뒤 업데이트 알림이 떠야 합니다.
- **1.0.3**은 자동 업데이트 DLL이 빠져 있어 알림이 나오지 않을 수 있습니다. 1.0.4 이상을 먼저 설치해 주세요.
