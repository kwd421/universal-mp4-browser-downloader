# ClipFlow 1.0.7

## 변경 사항

### 다운로드 / 구간
- **풀 다운로드 중 구간을 넣어도** 원래 카드가 바뀌지 않고, 구간은 **새 카드**로 받습니다.
- 분석 중·자동 다운로드 직전에 구간을 바꿔도, **누른 순간의 의도(풀/구간)를 고정**합니다.
- 완전 중복이 아니면 새 받기를 우선하고, 이미 받는 중/대기 중인 같은 대상은 막습니다.

### 이미 있는 파일
- 이미 받아 둔 파일이 있으면 **원래 완료 카드**로 스크롤하고 알림을 띄웁니다.
- 새로 생긴 빈 분석 카드에 중복 알림이 붙지 않도록 정리합니다.
- 중복 알림 애니메이션을 **검은 링 2번 깜빡임 + 같은 박자 배경 워시**로 다듬었습니다.

### 치지직
- 브라우저 쿠키를 너무 많이 실어 API가 **HTTP 400**을 내던 문제를 고쳤습니다.
- 네이버 로그인 쿠키(NID_AUT / NID_SES 등)만 넘기도록 필터합니다.

### 업데이트 알림
- 토스트에 **버전 번호**와 **상세**를 표시합니다.
- **×**로 닫으면 이번 실행에서만 숨기고, 앱을 다시 켜면 또 뜹니다.
- **상세**는 패치노트 링크를 엽니다.
