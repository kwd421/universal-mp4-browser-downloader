# ClipFlow 1.0.6

## 변경 사항

- 시작할 때 **응답 없음**이 뜨던 문제를 수정했습니다. 업데이트 확인 결과를 메인 스레드로 안전하게 넘기도록 고쳤습니다.
- WinSparkle DLL은 **업데이트** 버튼을 누를 때만 불러옵니다.
- 다운로드 기록은 백그라운드에서 읽고, 시작 직후 목록에 표시합니다.
- 업데이트 알림 토스트에서 **업데이트** 버튼 옆 불필요한 여백을 줄였습니다.
- 후보 선택·렌더 재진입·히스토리 프라이버시·URL 정리·스트림 fetch 안정성을 보강했습니다.
- 완료된 항목에서 파일이 없어도 삭제 버튼이 보이도록 수정했습니다.
- 완료 항목 더보기 메뉴의 구간 추출 창 제목과 확인 버튼 문구를 실제 동작에 맞게 수정했습니다.

## 확인 방법

1.0.5를 실행하면 시작 후 1~2초 뒤 **「새 버전이 있습니다」** 토스트가 뜹니다.
